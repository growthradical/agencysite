# Sticker scans go here

Drop your scanned/saved sticker images in this folder using **exactly** these
filenames. Any that are missing show up as a labelled placeholder in the album,
so you can print a draft at any point.

## Notes
- The roster above is a starting point based on the Yugoslavia Euro 92 squad.
  Confirm it against your own album and edit the list at the top of
  `album.html` if your section differs (names, numbers, or count).
- Portraits look best cropped to a ~5:7 (portrait) ratio; the hero to ~16:7.
- Where to find the original scans for reference:
  laststicker.com → "Panini UEFA Euro Sweden 1992", and tcdb.com.
