# PAS — Magazine of Macedonian Football

**PAS** (Cyrillic: **ПАС**, English: "the pass") is a digital-first and print
football magazine covering the game in North Macedonia — the national team, the
domestic 1.MFL, clubs in Europe, the global diaspora, history, culture and the
next generation.

> **Tagline:** *The story of Macedonian football, in full.*
> **Cyrillic tagline:** *Целата приказна на македонскиот фудбал.*

This repository holds the editorial and production plan for **Issue 01 (July
2026)** plus the launch social campaign. Everything here is written so a small
team (editor, designer, photo/AI-image lead, social manager) can execute it.

## How to use the image prompts

Every visual brief contains a **GPT image prompt** engineered for
GPT-image / DALL·E-class models. Two rules make them work:

1. **Generate the artwork text-free.** Image models render lettering badly.
   Each prompt asks for *clean negative space* where the masthead, headlines and
   captions are added afterwards in InDesign / Canva / Figma.
2. **Keep the house style block.** Paste the shared style block (see the prompt
   library) ahead of any scene description so every PAS image looks like it
   belongs to the same magazine.

## Editorial principles

- **Bilingual.** Macedonian first, English second — the diaspora (Australia,
  USA, Germany, Switzerland, Italy) is half the audience.
- **Honest.** Issue 01 lands weeks after the World Cup play-off heartbreak. We
  do not paper over it; we explain it and look forward.
- **Macedonian-made.** Local writers, local photographers, local design.
